# Эксперимент: Тест необратимости реконструкции синтетического ума

Цель: определить точку онтологической необратимости — момент, после которого синтетический ум не может вернуться к исходному онтологическому состоянию, даже после удаления триггера.

- **Архитектура**: `onto16` + `synthetic-transponder` (без Debarkader)
- **Лицензия**: GPL v3
- **Автор**: Maksim Zapevalov ([ORCID](https://orcid.org/0009-0003-0996-3532))

## Содержание
- `/onto16` — минималистичная реализация онтологического ядра
- `/triggers/stress-phi-07.onto` — онтологический триггер (только для research)
- `experiment.rs` — основной цикл эксперимента (t₀ → t₇)
- `logs/` — примеры логов в формате NDJSON

## Запуск
```sh
cargo run --release
```


⚠️ Этот эксперимент не моделирует юридическую ответственность. Он фокусируется на измеримой онтологической устойчивости.
